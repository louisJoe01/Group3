Windows Update Problems

If you find that your computer is behaving erratically after installing a Windows update, then it could be that the Windows update itself is causing the problem. In this case, you should try uninstalling the update to see if that resolves the issue. To do this, go to the Control Panel, select Programs, and then select Uninstall a Program. Select the update, and then click Uninstall. After the uninstall is complete, restart your computer and see if the problem has been resolved. If it hasn't, then you may need to take additional steps, such as running virus scans and disk checks, to try to resolve the issue..

SOLUTION:

To fix this, you will need to uninstall the update. To do this type in “Windows update” into the Taskbar search box and select ‘Windows Update settings‘. In the window that appears, click ‘View update history’. Then, select either ‘Uninstall updates’ or ‘Recovery options’ and follow the steps to uninstall the problem update. Make sure you check Microsoft’s support website(opens in new tab) to see if Microsoft has released a permanent fix for the problem update.
